# Politics-Games

## 選挙ゲーム - あなたの一票が未来を変える！

若者が選挙に参加したくなるような投票シミュレーションゲームです。

### 🎮 ゲームについて

このゲームでは、架空の町の選挙に参加します。比例代表制で様々な政党から選び、5つのランダムなラウンドを通じて投票の重要性を体験できます。

### 🎯 ゲームの特徴

- **8つの多様な政党**: 進歩派、若者重視、環境保護、中道、福祉、保守、そして危険な政党も...
- **12種類のイベント**: 毎回5つがランダムに選ばれます
- **20種類の豊富なエンディング**: あなたの選択によって異なる結末が待っています
- **リアルな選挙体験**: 投票率によって結果が変わる
- **危険な政党システム**: 投票しないと悪い政党が台頭することも
- **攻略性のあるゲームデザイン**: 特定のイデオロギーに一貫して投票することで専用エンディングを見られる

### 🎲 政党システム

#### 良い政党（6種類）
- **未来進歩党**: IT産業育成と若者雇用
- **若者第一党**: 若者の政治参加促進
- **環境保護党**: 自然保護と持続可能性
- **中道改革党**: バランスの取れた政策
- **福祉充実党**: 医療と福祉の充実
- **伝統保守党**: 伝統文化と財政健全化

#### 危険な政党（2種類・10%の確率で出現）
- **利権優先党**: 既得権益の保護と情報統制
- **強権統制党**: 監視社会と個人の自由の制限

### 🏆 エンディングシステム

合計20種類のエンディングが存在します！

#### 完全一貫エンディング（全5票を同じイデオロギーに）
- **ED1: テクノロジー革命** - 未来進歩党に5回投票
- **ED2: 若者革命** - 若者第一党に5回投票
- **ED3: エコトピアの実現** - 環境保護党に5回投票
- **ED4: 黄金の中道** - 中道改革党に5回投票
- **ED5: 福祉国家の完成** - 福祉充実党に5回投票
- **ED6: 伝統の継承** - 伝統保守党に5回投票

#### 強一貫エンディング（4票を同じイデオロギーに）
- **ED7: 革新の道** - 未来進歩党に4回投票
- **ED8: 若者の時代** - 若者第一党に4回投票
- **ED9: 緑の未来** - 環境保護党に4回投票
- **ED10: 調和の社会** - 中道改革党に4回投票
- **ED11: 優しい社会** - 福祉充実党に4回投票
- **ED12: 秩序ある発展** - 伝統保守党に4回投票

#### 多様性エンディング
- **ED13: 多様性の尊重/民主主義の実践/揺らぐ信念** - 複数のイデオロギーに投票

#### 消極的エンディング
- **ED14: 不完全燃焼** - 2回のみ投票
- **ED15: 消極的参加** - 1回のみ投票
- **ED16: 完全無関心** - 一度も投票せず

#### 危険なエンディング
- **ED17: 危機一髪** - 危険な政党に1回投票
- **ED18: 暗黒時代の到来** - 複数の危険な政党に投票
- **ED19: 腐敗の蔓延** - 利権優先党に3回以上投票
- **ED20: ファシズムの台頭** - 強権統制党に3回以上投票

### 🎯 ゲームの目的

- 投票の重要性を理解する
- 投票しないことの影響を学ぶ
- 若者の声が政治に与える影響を体感する
- 実際の選挙に参加したくなる
- **様々なエンディングを目指してリプレイする攻略性**

### 🚀 遊び方

1. `index.html` をブラウザで開く
2. 「ゲームを始める」をクリック
3. 各ラウンドで政党の政策を読む
4. 投票するか、投票しないかを選ぶ
5. 結果を見て、あなたの選択がどんな影響を与えたか確認する

### 💡 学べること

- **投票率の重要性**: 投票率が低いと、少数の人の意見だけで政治が決まってしまう
- **一票の価値**: 「一票くらい」と思う人が多いと、大きな差になる
- **若者の参加**: 若者が投票しないと、若者向けの政策が実現しにくい
- **社会への参加**: 選挙に参加することで、社会を変えることができる
- **危険な政党**: 無関心は危険な勢力の台頭を招く可能性がある
