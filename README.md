# Politics-Games

## 選挙ゲーム - あなたの一票が未来を変える！

若者が選挙に参加したくなるような投票シミュレーションゲームです。

### 🎮 ゲームについて

このゲームでは、架空の町の町長選挙に参加します。3つのラウンドを通じて、さまざまな政策課題について候補者を選び、投票の重要性を体験できます。

### 🎯 ゲームの目的

- 投票の重要性を理解する
- 投票しないことの影響を学ぶ
- 若者の声が政治に与える影響を体感する
- 実際の選挙に参加したくなる

### 🚀 遊び方

1. `index.html` をブラウザで開く
2. 「ゲームを始める」をクリック
3. 各ラウンドで候補者の政策を読む
4. 投票するか、投票しないかを選ぶ
5. 結果を見て、あなたの選択がどんな影響を与えたか確認する

### 💡 学べること

- **投票率の重要性**: 投票率が低いと、少数の人の意見だけで政治が決まってしまう
- **一票の価値**: 「一票くらい」と思う人が多いと、大きな差になる
- **若者の参加**: 若者が投票しないと、若者向けの政策が実現しにくい
- **社会への参加**: 選挙に参加することで、社会を変えることができる
