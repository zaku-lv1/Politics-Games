# Politics-Games

## 選挙ゲーム - あなたの一票が未来を変える！

若者が選挙に参加したくなるような投票シミュレーションゲームです。

### 🎮 ゲームについて

このゲームでは、架空の町の選挙に参加します。比例代表制で様々な政党から選び、3つのランダムなラウンドを通じて投票の重要性を体験できます。

### 🎯 ゲームの特徴

- **8つの多様な政党**: 進歩派、若者重視、環境保護、中道、福祉、保守、そして危険な政党も...
- **12種類のイベント**: 毎回3つがランダムに選ばれます
- **複数のエンディング**: バッドエンド、ノーマルエンド、グッドエンド、ハッピーエンド
- **リアルな選挙体験**: 投票率によって結果が変わる
- **危険な政党システム**: 投票しないと悪い政党が台頭することも

### 🎲 政党システム

#### 良い政党（6種類）
- **未来進歩党**: IT産業育成と若者雇用
- **若者第一党**: 若者の政治参加促進
- **環境保護党**: 自然保護と持続可能性
- **中道改革党**: バランスの取れた政策
- **福祉充実党**: 医療と福祉の充実
- **伝統保守党**: 伝統文化と財政健全化

#### 危険な政党（2種類・10%の確率で出現）
- **利権優先党**: 既得権益の保護と情報統制
- **強権統制党**: 監視社会と個人の自由の制限

### 🏆 エンディングシステム

- **ハッピーエンド**: 全ラウンドで良い政党に投票
- **グッドエンド**: 2回以上投票
- **ノーマルエンド**: 1回だけ投票
- **バッドエンド**: 投票しないか、危険な政党に投票

### 🎯 ゲームの目的

- 投票の重要性を理解する
- 投票しないことの影響を学ぶ
- 若者の声が政治に与える影響を体感する
- 実際の選挙に参加したくなる

### 🚀 遊び方

1. `index.html` をブラウザで開く
2. 「ゲームを始める」をクリック
3. 各ラウンドで政党の政策を読む
4. 投票するか、投票しないかを選ぶ
5. 結果を見て、あなたの選択がどんな影響を与えたか確認する

### 💡 学べること

- **投票率の重要性**: 投票率が低いと、少数の人の意見だけで政治が決まってしまう
- **一票の価値**: 「一票くらい」と思う人が多いと、大きな差になる
- **若者の参加**: 若者が投票しないと、若者向けの政策が実現しにくい
- **社会への参加**: 選挙に参加することで、社会を変えることができる
- **危険な政党**: 無関心は危険な勢力の台頭を招く可能性がある
